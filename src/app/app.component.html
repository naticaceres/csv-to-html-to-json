Upload CSV and convert to JSON Example

This example is meant to allow a user to upload a CSV file, display it in a table, rename headers and create a json object that could be sent to a backend.

What this example does not include is proper checking of csv file, dealing with commas inside of strings, or other more complex situations with a csv import. It assumes the first row of data will contain an element for every column, if it does not, there will be some unexpected behavior
